# Frontend Additional Dependencies

## Current Dependencies (Already Installed)
- react
- react-dom
- react-router-dom
- axios
- lucide-react
- tailwindcss
- @tailwindcss/vite

## Additional Dependencies Needed

### Data Visualization
```
npm install recharts
```
- For analytics charts, sales graphs, trust score visualization

### Voice/Audio
```
npm install react-media-recorder
```
- For audio recording in ChatAssistant

### Notifications
```
npm install react-hot-toast
```
- For toast notifications throughout the app

### Date Handling
```
npm install date-fns
```
- For date formatting in orders, transactions

### Form Handling
```
npm install react-hook-form zod @hookform/resolvers
```
- For better form validation and management

### File Upload
```
npm install react-dropzone
```
- For product image uploads

### Additional UI Components (Optional)
```
npm install @radix-ui/react-dialog
npm install @radix-ui/react-select
npm install @radix-ui/react-tabs
```
- For more complex UI components

### Testing (Future)
```
npm install --save-dev @testing-library/react @testing-library/jest-dom vitest
```
- For unit testing components

## Installation Commands

Install all additional dependencies:
```bash
npm install recharts react-media-recorder react-hot-toast date-fns react-hook-form zod @hookform/resolvers react-dropzone
```

Install dev dependencies:
```bash
npm install --save-dev @testing-library/react @testing-library/jest-dom vitest
```
